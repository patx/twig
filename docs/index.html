Twig

An ultra-minimal GTK code editor for Debian/Ubuntu based Linux distros.

One file per window. No toolbar. No menu bar. View screenshot.

Install

sudo apt install python3-gi gir1.2-gtk-3.0 gir1.2-gtksource-4
git clone https://gitman.io/git/patx/twig
cd twig
sudo make install

Features

  • Syntax highlighting
  • Line numbers
  • Find & replace
  • Undo / redo
  • Auto-indent
  • Dirty indicators
  • Jump to line
  • Print support
